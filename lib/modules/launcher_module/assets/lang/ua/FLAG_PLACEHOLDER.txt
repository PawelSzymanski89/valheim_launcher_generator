# Placeholder для українського прапора

Ця папка повинна містити файл `flag.png` (рекомендований розмір: 32x32 або 64x64 пікселі), що представляє прапор України.

Ви можете завантажити безкоштовні іконки прапорів з:
- https://flagicons.lipis.dev/
- https://www.flaticon.com/packs/countrys-flags
- https://github.com/lipis/flag-icons

Щоб додати прапор:
1. Завантажте PNG-зображення прапора України
2. Змініть розмір на 32x32 або 64x64 пікселі
3. Назвіть його `flag.png`
4. Помістіть у цю папку
5. Перезавантажте launcher або перезапустіть

Якщо файл flag.png відсутній, мова все одно працюватиме, але без іконки прапора в інтерфейсі.
